# 样本语料结构化描述

本文档提供5G核心网产品文档的结构化描述，用于指导大模型理解文档风格并生成相似语料。

---

## 一、文档类型概览

| 类型 | 目录 | 文档数量 | 核心特征 |
|------|------|----------|----------|
| 技术概念 | 5G基础知识 | ~97 | What/Why/How结构、图表丰富、概念解释详细 |
| 术语定义 | 快速入门 | ~5,521 | 属性表格式、三栏结构(名称/缩略语/定义) |
| CLI命令参考 | OM参考 | ~16,113 | 六段式结构(功能/格式/参数/视图/指南/实例) |
| 告警文档 | 网络运维 | ~804 | 八段式结构(名称/解释/属性/参数/影响/原因/处理/实例) |
| 解决方案概述 | 特性部署 | ~1,874 | 方案介绍+业务流程+配置说明 |
| 部署指南 | 网络部署 | ~306 | 步骤式操作、配置示例、验证方法 |

## 二、详细结构说明

### 2.1 技术概念文档 (5G基础知识)

**示例文件**: `网络切片_42265154.md`, `QoS_42268275.md`

**章节结构**:
1. 概念引入（What is X）
2. 技术原理（Why need X）
3. 实现机制（How to implement X）
4. 相关参数/配置
5. 流程图/架构图
6. 表格对比

**格式特征**:
- 标题层级清晰（H1→H2→H3）
- 大量使用架构图、流程图
- 关键概念使用加粗/斜体标注
- 参数/属性使用表格呈现

---

### 2.2 术语定义文档 (快速入门)

**示例文件**: `AMBR_41972681.md`, `PCF_41972749.md`

**章节结构**:
```
# 术语名称 (H1)

## 缩略语
XXX

## 定义
详细定义说明...
```

**格式特征**:
- 简洁的三栏式结构
- 术语名称作为H1标题
- 缩略语单独成行
- 定义说明包含2-3句话解释

---

### 2.3 CLI命令参考 (OM参考)

**示例文件**: `地址池视图命令\section_60796051.md`

**章节结构**:
1. **命令功能**: 简要说明命令用途
2. **命令格式**: `COMMAND param1 param2 [optional]`
3. **参数说明**: 表格形式(参数名/类型/取值范围/缺省值/说明)
4. **视图**: 进入该命令的视图层级
5. **使用指南**: 使用前提条件和注意事项
6. **使用实例**: 完整的配置示例

**格式特征**:
- 命令关键字使用**加粗**
- 参数使用`等宽字体`
- 实例包含完整输入输出
- 使用竖线表格呈现参数

---

### 2.4 告警文档 (网络运维)

**示例文件**: `ALM-100001 进程故障告警_70996391.md`

**章节结构**:
1. **告警名称**: `ALM-XXXXXX 告警名称`
2. **告警解释**: 告警触发条件的文字描述
3. **告警属性**: 表格(告警ID/告警级别/生成周期/是否可恢复)
4. **告警参数**: 表格(参数名/参数含义)
5. **对系统的影响**: 告警对业务的影响说明
6. **可能原因**: 列表形式(原因1/原因2/...)
7. **处理步骤**: 编号列表形式的具体处理流程

**格式特征**:
- 告警ID使用特定格式 `ALM-XXXXXX`
- 告警级别: 紧急/重要/次要/提示
- 处理步骤使用有序列表
- 包含"建议"和"注意事项"提示框

---

### 2.5 解决方案概述 (特性部署)

**示例文件**: `5GC国际漫游解决方案概述_11244470.md`, `方案介绍_72546024.md`

**章节结构**:
1. **方案概述**: 解决方案的总体介绍
2. **What/Why**: 问题定义和业务背景
3. **技术原理**: 实现机制说明
4. **组网架构**: 网络拓扑和接口
5. **业务流程**: 典型流程图和说明
6. **配置部署**: 具体配置步骤

**格式特征**:
- 使用"图X"格式标注图片
- 对比项使用表格(如HR vs LBO架构对比)
- 流程步骤使用编号列表
- 关键概念使用加粗标注

---

### 2.6 部署指南 (网络部署)

**示例文件**: `熟悉网络架构_75096814.md`, `公共参数配置_97905321.md`

**章节结构**:
1. **概述**: 配置任务简介
2. **前提条件**: 配置前的准备工作
3. **配置流程**: 步骤式配置说明
4. **配置参数**: 参数表格
5. **验证方法**: 配置后的验证步骤

**格式特征**:
- 操作步骤使用有序列表
- 配置示例使用代码块
- 警告/注意使用提示框
- 包含"相关配置"链接

## 三、格式规范总结

### 3.1 标题层级
- H1: 文档主题/章节标题
- H2: 二级标题(功能模块)
- H3: 三级标题(具体特性)

### 3.2 表格使用
- 参数说明使用竖线表格
- 对比分析使用横线表格
- 属性列表使用属性表格

### 3.3 图表标注
- 图片格式: `**图X**图注说明`
- 流程图用于业务流程
- 架构图用于网络拓扑

### 3.4 格式标记
- **加粗**: 关键字、命令名、参数名
- `等宽字体`: 命令语法、参数值、代码
- 有序列表: 步骤流程
- 无序列表: 要点列举
