# 大模型语料生成提示词模板

基于UDG 5G核心网产品文档的结构化描述，为大模型提供生成相似语料的指导。

---

## 任务说明

你需要根据以下提供的文档结构规范和内容背景，生成一篇符合UDG产品文档风格的markdown格式文档。

## 内容生成示例

### 示例：生成一篇"SMF会话管理"术语定义文档

**输入**:
- 文档类型: 术语定义文档
- 主题: SMF (Session Management Function)

**输出**:

```markdown
# SMF

## 缩略语
Session Management Function

## 定义
SMF是5G核心网中的会话管理功能网元，负责用户PDU会话的建立、修改和释放。SMF通过与AMF交互获取会话建立请求，与PCF交互获取策略控制信息，选择合适的UPF作为会话锚点，并负责N4会话管理以及UPF资源分配。SMF还支持与4G PGW-C的融合部署，实现4G/5G互操作场景下的会话管理统一控制。
```

## 格式规范提醒

1. **语言**: 全部使用中文，包括标题、正文、表格内容
2. **格式**: 使用标准Markdown格式
3. **术语**: 首次出现的中文术语后附加英文原文
4. **图表**: 使用"图X"格式标注，表格使用Markdown表格语法
5. **命令**: 使用等宽字体显示命令和参数
6. **列表**: 步骤使用有序列表，要点使用无序列表
```

---

## 使用说明

1. 根据需要生成的文档类型，选择对应的模板
2. 填充"基础信息"部分
3. 按照模板结构填充内容
4. 注意遵循格式规范
5. 确保技术内容准确专业
