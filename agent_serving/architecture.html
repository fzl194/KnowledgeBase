Agent Serving 架构

在线知识检索层。面向 Skill / Agent 提供 HTTP API，从 Mining 生成的知识资产中读取唯一 active release，通过 FTS5 BM25 + Graph BFS 检索，返回结构化 ContextPack。不做解析、不写数据库、不生成自然语言答案。

05

Graph BFS 图扩展

基于 raw_segment_relations 的广度优先遍历，双向查找邻居。

BFS 算法
visited = set(seed_segment_ids)
frontier = list(seed_segment_ids)      ← seed: FTS5 检索命中的 segments

for depth in range(1, max_depth + 1):  ← default max_depth=2
    neighbors = get_neighbors(frontier) ← 双向: source→target AND target→source
    for neighbor in neighbors:
        if neighbor not in visited:     ← 防环
            visited.add(neighbor)
            expanded.append(depth, relation_type, from_id)
    frontier = new_neighbors
    if len(expanded) >= max_results:    ← default max=20, 防爆炸
        break
参数默认值说明
max_depth2最大 BFS 深度。depth=1 为直接邻居，depth=2 为二跳
max_results20最大扩展结果数。防止关系密集区域爆炸
relation_types全部可选过滤：previous/next/same_section/section_header_of
遍历方向双向同时查找 source→target 和 target→source，确保不遗漏
范围约束snapshot_ids只返回 active build 范围内的 segment
